

#  Blinkit Sales & Customer Insights Dashboard 

## 📌 Project Overview

This project presents an interactive Power BI dashboard developed to analyze Blinkit's sales performance, outlet efficiency, customer purchasing behavior, and product trends.

The dashboard enables users to explore business performance through dynamic visualizations, KPIs, filters, and drill-through pages for detailed analysis.

---

# 🎯 Objectives

- Analyze overall sales performance
- Identify top-performing products and outlets
- Understand customer purchasing patterns
- Compare outlet performance
- Analyze item visibility and ratings
- Generate actionable business insights

---

# 📂 Dataset

The dataset contains Blinkit grocery sales information including:

- Item Identifier
- Item Type
- Item Fat Content
- Outlet Type
- Outlet Location
- Outlet Establishment Year
- Item Visibility
- Item Weight
- Sales
- Ratings

# 📈 Key Business Insights

✅ Total Sales exceeded **1.20 Million**

✅ Fruits & Vegetables generated the highest sales.

✅ Snack Foods were among the top-selling categories.

✅ Tier 3 outlets contributed the highest revenue.

✅ Supermarket Type 1 showed the best overall sales performance.

✅ Regular Fat items generated higher sales than Low Fat items.

✅ Customer ratings remained consistently high across products.
